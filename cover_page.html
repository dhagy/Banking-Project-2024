CSCI 225 Final Project

Title: Banking Website

Team Members: Dylan (Team Leader), Edgar

Project Information

Description:

This project is a simulated banking application that includes essential functionalities such as user sign-in, account management, fund transfers, deposits, and customer support. The system is designed with HTML, CSS, and JavaScript, utilizing modern web development practices for an intuitive user interface and secure database integration.

Key Features:

  • Google Sign-In for secure authentication.
  • Account management: Check balances, view transaction history.
  • Deposit page for adding funds to the account.
  • Transfer functionality with recipient verification.
  • Customer support page for resolving issues.

Rubric Details

  • HTML Pages: Includes cover page, sign-in/homepage, account management page, transfer and deposit pages, and customer support page.
  • CSS: Consistent and professional styling, with a focus on layout and typography.
  • JavaScript: Interactive elements for transferring funds, depositing funds, and managing account data.
  • Database: Google Sign-In and storage of user transaction history and balances.
